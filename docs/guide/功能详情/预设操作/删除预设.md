## deletePreset

删除 `preset_name` 预设。

```typescript
async function deletePreset(preset_name: Exclude<string, 'in_use'>): Promise<boolean>;
```

### 参数

#### preset_name

- **类型**: `string`
- **描述**: 预设名称

### 返回值

- **是否成功删除, 可能因预设不存在等原因而失败**: `boolean`
