## substitudeMacros

替换字符串中的 SillyTavern 宏。

```typescript
async function substitudeMacros(text: string): Promise<string>;
```

### 参数

#### `text`

- **类型**: `string`
- **描述**: 需要替换的字符串

### 返回值

- **替换结果**: `string`

### 示例

```typescript
const text = substitudeMacros("{{char}} speaks in {{lastMessageId}}");
text == "少女歌剧 speaks in 5";
```
