## getCharacterNames

获取角色卡名称列表

```ts
function getCharacterNames(): string[];
```

### 返回值

- **角色卡名称列表**: `string[]`

## getCurrentCharacterName

```ts
function getCurrentCharacterName(): string | null;
```

### 返回值

- **角色卡名称**: `string|null`; 如果当前没有角色卡, 则返回 `null`

## getCharacter

获取 `character_name` 角色卡的内容

:::code-group

```ts [getCharacter]
async function getCharacter(character_name: LiteralUnion<'current', string>): Promise<Character | null>;
```

```ts [Character]
ttype Character = {
  name: string;
  version: string;
  creator: string;

  extensions: {
    regex_scripts: TavernRegex[];

    };
    [other: string]: any;
  };
};
```

:::

### 参数

#### character_name

- **类型**: `string`
- **描述**: 角色卡名称

### 返回值

- **角色卡内容**: [`Character`](#getcharacter)

### 异常

- 如果角色卡不存在, 将会抛出错误
