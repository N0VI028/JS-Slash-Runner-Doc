## importRawPreset

像酒馆界面里那样导入新预设/更新现有预设

:::code-group

```ts [importRawPreset]
function importRawPreset(filename: string, content: string): Promise<boolean>;
```

:::

### 参数

#### `filename`

- **类型**: `string`
- **描述**: 预设文件名

#### `content`

- **类型**: `string`
- **描述**: 预设文件内容

### 示例

:::code-group

```ts [importRawPreset]
const response = await fetch(预设网络链接);
await importRawPreset(预设名, await response.text());
```

:::
